The majestic Himalayas of Nepal featuring ancient temples, snow-capped peaks, prayer flags, and Kathmandu valleys. Explore the beauty of this mountain kingdom through this photo gallery and see how easy it is to replicate with some MDX magic and tailwind classes.

Features images served using `next/image` component. The locally stored images are located in a folder with the following path: `/static/images/nepal/[filename].jpg`

Since we are using mdx, we can create a simple responsive flexbox grid to display our images with a few tailwind css classes.
